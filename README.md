# Library-Management-System-Django-CRUD-App-
A web-based Library Management System built using Django that allows librarians to manage books, members, and borrow/return transactions efficiently.

## Features
- Add Books, Members, Authors, Book Categories
- Edit Books, Members, Authors, Book Categories
- Delete Books, Members, Authors, Book Categories
- View Books list, Members list, Authors list, Book Categories list, Borrowers List
- Return Book, Borrow Book
- User login/logout
